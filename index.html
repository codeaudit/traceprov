About

Java library for provenance management:

  • Holds fully documented Java models for DCAT catalog vocabulary
  • Allows references to dataset elements such as cells, columns, etc
  • All classes are immutable

  • OpenDataRise: power tool to cleanse and semantify open data, based on OpenRefine
